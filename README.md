# BranchChat

**An AI chat app with branching conversations — explore multiple threads from any message, all inside a single window.**

Unlike linear chat interfaces (ChatGPT, Gemini, Perplexity), BranchChat lets you **fork** the conversation at any point, creating a tree of related discussions. Switch between branches instantly without losing context.

## ✨ Features

- **Branching conversations** – Turn any message into a new thread.
- **Non‑linear navigation** – Jump between branches inside the same window.
- **Full chat history** – Each branch retains its own context.
- **AI model agnostic** – Works with OpenAI, Anthropic, Gemini, or local LLMs (bring your own API key).
- **Light/dark mode** – Because eyes matter.
